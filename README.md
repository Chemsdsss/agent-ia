# LIA – Agent Codeur IA

Bug ? discord "pas2profil"

LIA est un assistant IA en ligne de commande qui **code, teste et corrige** automatiquement vos projets.  
Il peut aussi rechercher sur le web, gérer GitHub, lire l’actualité, et bien plus.

**Double mode** :  
- ☁️ API Groq (gratuit, avec limites)  
- 🖥️ Local via Ollama (hors‑ligne, illimité)

## 🚀 Installation

```bash
git clone https://github.com/Chemsdsss/agent-ia.git
cd agent-ia
pip install -r requirements.txt
python setup.py   # assistant de configuration (choix du mode)
